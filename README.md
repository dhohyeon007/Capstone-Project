# 졸업 프로젝트: 멀티모달 문서 이해 기반 보고서 자동 데이터 구조화 시스템

## 📚 프로젝트 개요

본 프로젝트는 정부 보고서와 같이 차트, 표, 그림, 그래프 등 다양한 멀티미디어 요소가 혼재된 비정형 한글(HWP) 문서를 자동으로 이해하고, 핵심 정량 정보를 추출하여 구조화된 데이터 형태로 변환하는 인공지능 시스템을 개발하는 것을 목표로 합니다. 수많은 공공 문서로부터 유의미한 데이터를 효율적으로 추출하고 활용하기 위한 실무형 AI 파이프라인을 구축하여, 데이터 기반 의사결정을 지원하는 데 기여하고자 합니다.

## 🎯 주요 목표

- **멀티모달 문서 이해:** 한글(HWP) 문서를 입력받아 텍스트, 표, 차트, 그림 등 다양한 구성 요소를 정확히 인식하고 분석합니다.
- **핵심 정보 추출 및 구조화:** 문서 내의 핵심 정량 정보를 추출하고, 엑셀과 같은 스프레드시트 형태로 구조화된 데이터베이스를 구축합니다.
- **실무형 AI 파이프라인 구축:** 실제 공공 문서에서 발생하는 난독성 및 비정형성 문제를 해결할 수 있는 견고하고 실용적인 AI 시스템을 개발합니다.

## ⚙️ 핵심 기능 및 개발 단계

본 시스템은 다음과 같은 핵심 기능들을 단계적으로 구현하여 구성됩니다.

1.  **텍스트 인식 (OCR) 모듈:**
    - HWP 문서 내의 텍스트를 정확하게 인식하고 추출합니다.
    - 다양한 글꼴, 크기, 서식에 대응할 수 있도록 설계합니다.
2.  **레이아웃 분석 모듈:**
    - 문서의 전반적인 구조(단락, 제목, 목록 등)와 각 요소(텍스트 블록, 이미지, 표, 차트)의 위치 관계를 파악합니다.
    - HWP 문서 특유의 복잡한 레이아웃을 이해하는 데 중점을 둡니다.
3.  **표/차트 해석 모듈:**
    - 문서 내의 표와 차트를 자동으로 감지하고, 그 안에 담긴 데이터를 추출 및 해석합니다.
    - 차트의 종류(막대, 선, 원형 등)를 식별하고 시각적 정보를 수치 데이터로 변환합니다.
4.  **의미 기반 정보 정렬 모듈:**
    - 추출된 텍스트, 표, 차트 데이터를 통합하여 의미론적으로 연결하고, 일관된 구조의 정보로 재정렬합니다.
    - 도메인 특화된 지식을 활용하여 정보의 연관성을 판단하고 최종 구조화된 데이터를 생성합니다.

## 📦 예상 프로젝트 결과물

- **멀티모달 보고서 자동 데이터 구조화 AI 시스템:**
  - HWP 문서를 입력받아 구조화된 엑셀/CSV 형식의 데이터 파일을 출력하는 실행 가능한 소프트웨어.
  - 각 모듈(OCR, 레이아웃 분석, 표/차트 해석, 의미 기반 정렬)이 유기적으로 연동되는 파이프라인.
